# Numbers, Arithmetic, and Variables in Python

## Why This Matters

Before you build websites, automate servers, analyze data, write DevOps tools, or create applications, you must learn how Python works with numbers and how it stores information.

Every program eventually works with data:

- A shopping app calculates totals.
- A payroll system computes salaries.
- A monitoring tool calculates CPU usage percentages.
- A cloud automation script counts servers.
- A budgeting app tracks expenses.

All of these tasks rely on numbers.

However, numbers alone are not enough. Programs must also remember values. That is where **variables** come in.

Think of a variable as a labeled container.

```text
monthly_salary
      |
      v
   50000
```

Instead of repeatedly typing a value, you store it in a variable and reuse it whenever needed.

In this chapter you will learn:

- Integer numbers
- Floating-point numbers
- Arithmetic operators
- Division and floor division
- Remainders using modulo
- Powers and roots
- Order of operations
- Variable assignment
- Variable naming rules
- Common beginner mistakes

These concepts form the foundation of everything that follows in Python.

---

## Quick Example

```python
monthly_salary = 50000
tax_rate = 0.10

tax_amount = monthly_salary * tax_rate

print(tax_amount)
```

Output:

```text
5000.0
```

### What Happened?

1. Python stored `50000` inside `monthly_salary`.
2. Python stored `0.10` inside `tax_rate`.
3. It multiplied the values.
4. The result was stored in `tax_amount`.
5. `print()` displayed the result.

This simple example demonstrates both numbers and variables working together.

## Core Concepts

### What Are Numbers?

Numbers are pieces of data that represent quantities.

Python supports many numeric types, but beginners should first focus on:

1. Integers (`int`)
2. Floating-point numbers (`float`)

---

### Integers

Integers are whole numbers.

Examples:

```python
0
5
100
-10
2500
```

```python
print(type(100))
```

Output:

```text
<class 'int'>
```

---

### Floating-Point Numbers

Floating-point numbers contain decimal values.

Examples:

```python
3.14
10.5
-2.7
0.001
```

```python
print(type(3.14))
```

Output:

```text
<class 'float'>
```

---

### Scientific Notation

Python can represent large and small numbers using exponential notation.

```python
distance = 4e2

print(distance)
```

Output:

```text
400.0
```

Explanation:

```text
4e2

means

4 × 10²

= 400
```

Another example:

```python
print(3e3)
```

Output:

```text
3000.0
```

---

### What Are Variables?

Variables allow you to store data and give it a meaningful name.

```python
user_age = 25
```

Memory illustration:

```text
user_age
    |
    v
   25
```

Instead of remembering the number yourself, Python remembers it for you.

## Arithmetic Operators

| Operator | Meaning        | Example  |
| -------- | -------------- | -------- |
| `+`      | Addition       | `5 + 2`  |
| `-`      | Subtraction    | `5 - 2`  |
| `*`      | Multiplication | `5 * 2`  |
| `/`      | Division       | `5 / 2`  |
| `//`     | Floor Division | `5 // 2` |
| `%`      | Modulo         | `5 % 2`  |
| `**`     | Power          | `5 ** 2` |

## Division

### Normal Division

```python

```

Output:

```text
1.75
```

Python always returns a float when using `/`.

---

### Floor Division

```python
print(7 // 4)
```

Output:

```text
1
```

Floor division removes the decimal portion and rounds down toward negative infinity.

Visualization:

```text
7 ÷ 4

= 1.75

Floor Value

= 1
```

---

### Example

```python

capacity = 5

full_boxes = boxes // capacity

print(full_boxes)
```

Output:

```text
4
```

## Debugging Lab

### Broken Code

```python
monthly salary = 50000

print(monthly salary)
```

Error:

```text
SyntaxError: invalid syntax
```

---

### Root Cause

Variable names cannot contain spaces.

Python interprets:

```python
monthly salary
```

as two separate words.

---

### Fix

```python
monthly_salary = 50000

print(monthly_salary)
```

Output:

```text
50000
```

---

### Another Common Error

Broken Code:

```python
income = 50000

print(incom)
```

Error:

```text
NameError: name 'incom' is not defined
Did you mean: 'income'?
```

---

### Fix

```python
income = 50000

print(income)
```

Output:

```text
50000
```

## Common Beginner Mistakes

### Mistake 1: Using `=` Instead of Storing a Value

Incorrect:

```python
5 = age
```

Why Wrong:

Python expects the variable name on the left side.

Correct:

```python
age = 5
```

---

### Mistake 2: Using Spaces in Variable Names

Incorrect:

```python
user name = "Alice"
```

Correct:

```python
user_name = "Alice"
```

---

### Mistake 3: Forgetting Assignment

Incorrect:

```python
total + 5
```

Why Wrong:

The result is calculated but never stored.

Correct:

```python
total = total + 5
```

---

## Best Practices

### Use Descriptive Names

Good:

```python
monthly_income
```

Bad:

```python
x
```

---

### Use Lowercase With Underscores

Good:

```python
tax_rate
server_count
```

---

### Avoid Reserved Words

Bad:

```python
list = [1, 2, 3]
str = "hello"
```

Better:

```python
numbers = [1, 2, 3]
message = "hello"
```

---

### Keep Calculations Readable

Good:

```python
monthly_cost = server_cost * server_count
```

Bad:

```python
m = s * c
```

## Key Takeaways

- Python primarily uses integers (`int`) and floating-point numbers (`float`) for numeric calculations.
- Variables allow programs to remember and reuse data.
- Use `=` to assign values to variables.
- `/` performs regular division and returns a float.
- `//` performs floor division.
- `%` returns the remainder after division.
- `**` is used for powers and roots.
- Python follows standard mathematical order of operations.
- Variables can be reassigned at any time.
- Use descriptive, lowercase variable names with underscores.
- Understanding numbers and variables is essential because nearly every Python program depends on them.
